Round 3: Mysterious Round

Objective: Participants will analyze a provided C code snippet that involves a complex swapping mechanism. They will identify issues, propose modifications, and discuss how the corrected code should function.

How It Works:

Problem Statement: Participants are given a C code snippet with a swapping mechanism that they need to understand and correct.

Challenge Phases:

  • Analysis Phase (5 minutes): Participants read the code snippet and identify the swapping logic and any potential issues.
  • Modification Phase (5 minutes): On paper, participants write down the proposed changes to correct or optimize the code.
  • Explanation Phase (5 minutes): Participants explain their modifications and how they address the issues. They write a brief explanation of how the corrected code should work.
