Break The Query - Round 2

Round 2: Code Swap Challenge

Description: In this round, participants will be divided into teams. Each team will be given code for a searching or sorting algorithm that they need to work on.

Details: Teams will start with an initial code, and they must make changes based on instructions given by the organizers. At any point, the organizers may announce a "swap," requiring the teams to modify their code accordingly. This round will test participants' problem-solving and adaptability skills.

Rules:

  • Teams will be given specific time to write and implement their code.
